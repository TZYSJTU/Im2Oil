How to run our code:

1. Firstly, put the input image in the folder "input". We have offered 50 inputs, which can be seen in this folder.

2. Secondly, python Oil-painting.py --filename "./input/filename.jpg" --p_max xxxx 

	for example: python Oil-painting.py --filename "./input/Hepburn.jpg" --p_max 0.25
										(0<p_max<1)

3. The result will be in the folder "output".
